# 🌐✨ Desenvolvimento de Interfaces Web (DIW)

💻 Repositório dedicado ao desenvolvimento de um site autoral criado ao longo da disciplina de Desenvolvimento de Interfaces Web — PUC Minas

---

## 🌸 Sobre a disciplina

A disciplina de DIW tem como foco a criação de interfaces web modernas, combinando design, usabilidade e desenvolvimento.

Ao longo do semestre, foi desenvolvido um projeto completo de site, com o objetivo de representar identidade, criatividade e domínio das tecnologias web.

---

## 🎨 Sobre o projeto

O projeto consiste na construção de um site que reflita **quem eu sou**, minhas ideias e minha forma de pensar.

Mais do que um exercício técnico, é um espaço de expressão pessoal através da tecnologia.

---

## 🧠 Tecnologias utilizadas

* HTML
* CSS
* JavaScript

💡 Possíveis conceitos aplicados:

* Estruturação de páginas
* Estilização e responsividade
* Interatividade com JavaScript
* Organização de layout

---

## 🛠️ Desenvolvimento ao longo do semestre

O site foi evoluindo em etapas, incluindo:

* 📌 Planejamento da interface
* 📌 Criação da estrutura inicial
* 📌 Estilização visual
* 📌 Implementação de interações
* 📌 Ajustes e melhorias

---

## 🎯 Objetivo do repositório

* 📌 Registrar a evolução do projeto
* 📌 Organizar os arquivos do site
* 📌 Demonstrar habilidades em desenvolvimento web
* 📌 Servir como portfólio pessoal

---

## 💅 Destaques

✨ Projeto autoral

✨ Aplicação prática de conceitos web
✨ Evolução contínua ao longo do semestre

---

## 🌷 Observação final

Mais do que código, esse projeto representa uma construção criativa.

Cada detalhe do site foi pensado para unir estética e funcionalidade, mostrando que desenvolvimento web também é uma forma de expressão.
